# Security Policy

## Supported Versions

The following versions of Nexus Chat are currently supported with security updates:

| Version | Supported |
|--------|-----------|
| v1.x.x | ✅ Yes |
| < v1.0 | ❌ No |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

###  How to Report
- Use **GitHub Private Vulnerability Reporting**
- OR email us directly at: **admin@nexusainow.online** 

###  Do NOT
- Open a public GitHub issue
- Share details on social media or forums

We aim to acknowledge reports within **48 hours** and resolve critical issues as quickly as possible.

Thank you for helping keep Nexus Chat secure.
